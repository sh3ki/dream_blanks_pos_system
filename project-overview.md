# Dream Blanks POS System - Project Overview

## 1. Project Summary
**Dream Blanks POS (Point of Sale) System** is a comprehensive, web-based point-of-sale and inventory management solution designed specifically for retail businesses. It provides end-to-end capabilities for managing users, clients, products, inventory, sales transactions, and detailed reporting.

## 2. Business Context
- **Company**: Dream Blanks
- **Purpose**: Streamline retail operations, manage inventory, process sales, and track financial transactions
- **Target Users**: Retail staff, inventory managers, sales personnel, and administrators
- **Deployment Model**: SaaS (Software as a Service) on shared PHP hosting (Hostinger)

## 3. Core Objectives
1. **Simplify Sales Operations** - Fast and efficient POS checkout experience
2. **Inventory Control** - Real-time inventory tracking and restocking management
3. **Financial Tracking** - Comprehensive invoice generation, payment tracking, and reporting
4. **User Management** - Role-based access control with granular permissions
5. **Business Intelligence** - Advanced reporting and analytics dashboard
6. **Audit Trail** - Complete logging of all system actions for compliance and security
7. **Payment Flexibility** - Support for multiple payment modes (Cash, BDO, GCash)

## 4. Key Features at a Glance
- **User & Authentication**: Login with email/username, OTP-based password recovery
- **Role & Permission Management**: Dynamic permission assignment per role
- **Client Management**: Comprehensive client profiles with multiple addresses and contacts
- **Product Management**: Full product lifecycle with variants (color, size, category)
- **Inventory Management**: Stock tracking, low-stock alerts, restocking workflow
- **POS Module**: Modern point-of-sale interface with cart management
- **Invoice System**: Professional invoice generation with customizable templates
- **Payment Tracking**: Support for full, partial, and unpaid transactions
- **Transaction Logs**: Complete audit trail of all business transactions
- **Reports & Analytics**: Charts, statistics, and exportable reports
- **Dashboard**: Real-time business metrics and key performance indicators
- **Notifications**: Event-driven notifications throughout the system

## 5. Technology Stack
- **Backend**: PHP (Vanilla, OOP-based)
- **Database**: MySQL
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Hosting**: PHP Hostinger Shared Server
- **Architecture**: Modular, component-based OOP pattern

## 6. Design Philosophy
- **Minimalist Design**: Clean, professional, and modern interface
- **Color Scheme**: Gray and white theme for professionalism
- **Responsive**: Works seamlessly across desktop and tablet
- **Accessibility**: User-friendly navigation and intuitive workflows
- **Performance**: Optimized for shared hosting environment

## 7. Project Scope
### In Scope
- User management and authentication
- Role-based access control
- Complete inventory management
- POS functionality
- Invoice generation and tracking
- Financial reporting
- System-wide audit logging
- Dashboard and analytics

### Out of Scope (Future Enhancements)
- Mobile app (Phase 2)
- Advanced AI-based inventory predictions
- Multi-branch management
- Integration with accounting software

## 8. Success Criteria
- System goes live and is accessible on Hostinger server
- All core modules function as designed
- Users can complete full POS transactions end-to-end
- Comprehensive audit trail is maintained
- Reports can be generated and downloaded as CSV
- System handles 100+ concurrent users without performance degradation
- Data integrity is maintained across all modules
